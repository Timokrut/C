\doxysubsection{Detailed Description}
Implements the Shannon-\/\+Fano compression algorithm. 

The \doxylink{classCompressor}{Compressor} class provides functionality for analyzing text, generating codes for characters based on frequency, and encoding text using the Shannon-\/\+Fano algorithm.
