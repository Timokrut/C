\doxysubsection{Detailed Description}
Implements the decompression logic for Shannon-\/\+Fano compressed files. 

The \doxylink{class_decompressor}{Decompressor} class provides functionality to load a dictionary of character codes and decode an encoded string based on the Shannon-\/\+Fano algorithm.
